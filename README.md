# 🏐 Volley Pizol – Interne Dokumente

Interaktive digitale Dokumente für Spieler:innen und Trainer:innen von Volley Pizol.

## 📄 Inhalt

| Dokument | Beschreibung |
|---|---|
| [Trainingsphilosophie](volley_pizol_trainingsphilosophie.html) | Unsere 9 Grundprinzipien, Leitgedanke und Trainer:innen-Version als interaktives Poster |

Weitere Dokumente folgen.

## 🚀 Verwendung

Die Seite ist direkt im Browser aufrufbar – keine Installation nötig.  
Alle Dokumente sind vollständig interaktiv (Popups, Navigation etc.).

**Startseite:** `index.html`  
Öffne die Startseite und wähle das gewünschte Dokument aus.

## 📁 Dateistruktur

```
volley-pizol-trainingsphilosophie/
├── index.html                               # Startseite / Übersicht
└── volley_pizol_trainingsphilosophie.html   # Trainingsphilosophie & Grundprinzipien
```

## ➕ Neues Dokument hinzufügen

1. Neue HTML-Datei ins Repository laden
2. In `index.html` einen neuen Kachel-Block einfügen:

```html
<a class="card" href="dateiname.html">
  <div class="card-icon">📋</div>
  <div class="card-title">Titel</div>
  <div class="card-desc">Kurze Beschreibung des Dokuments.</div>
  <div class="card-link">Öffnen →</div>
</a>
```
