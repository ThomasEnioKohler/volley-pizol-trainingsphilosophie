🚀 Verwendung
Die Seite ist direkt im Browser aufrufbar – keine Installation nötig.
Alle Dokumente sind vollständig interaktiv (Popups, Navigation etc.).
Startseite: index.html
Öffne die Startseite und wähle das gewünschte Dokument aus.
📁 Dateistruktur
volley-pizol-trainingsphilosophie/
├── index.html                               # Startseite / Übersicht
└── volley_pizol_trainingsphilosophie.html   # Trainingsphilosophie & Grundprinzipien
